# 🌐 Portfólio - Antônio Victor

Este é o meu projeto de **portfólio pessoal**, desenvolvido para apresentar meus trabalhos, habilidades e trajetória acadêmica.  
O objetivo é ter uma vitrine simples e funcional para compartilhar no **GitHub** e **LinkedIn**.

---

## 🚀 Tecnologias utilizadas
- **HTML5** → Estrutura semântica das páginas  
- **CSS3** → Estilização responsiva e moderna  
- **JavaScript** → Interatividade e integração com WhatsApp  

---

## 🎨 Funcionalidades
- Navegação suave entre seções  
- Cabeçalho com foto e informações principais  
- Seção **Sobre mim** com resumo da trajetória  
- Seção **Projetos** com cards interativos e links diretos para repositórios  
- Formulário de contato integrado ao **WhatsApp**  
- Efeito visual de partículas no fundo para dar dinamismo
